# 业主小程序使用手册

## 1. 注册与登录

### 1.1 注册账号

1. 打开业主小程序，点击「注册」
2. 填写以下信息：
   - 手机号码（11 位）
   - 短信验证码（6 位数字，5 分钟有效）
   - 选择所属小区
   - 填写房屋号（如 `1-1-101`）
   - 身份证后 4 位
3. 点击「提交注册」
4. 注册后账号状态为「待审核」，物业审核通过后即可使用

**注意事项：**
- 验证码连续输错 3 次将锁定 10 分钟
- 同一房屋号支持多个业主注册（如夫妻共同使用）

### 1.2 登录

1. 输入注册手机号
2. 获取并输入短信验证码
3. 登录成功后自动跳转首页

## 2. 车牌管理

### 2.1 添加车牌

1. 进入「车牌管理」页面
2. 点击「添加车牌」
3. 填写车牌号、品牌、型号、颜色
4. 点击「确认添加」

**限制：** 每个房屋号最多绑定 5 个车牌

### 2.2 删除车牌

1. 在车牌列表中找到目标车牌
2. 点击「删除」
3. 确认删除

**注意：** 车辆在场时无法删除对应车牌

### 2.3 设置 Primary 车辆

1. 在车牌列表中选择目标车牌
2. 点击「设为 Primary」
3. 阅读提示信息，确认操作

**Primary 车辆说明：**
- 每个房屋号只能设置 1 辆 Primary 车辆
- Primary 车辆享有自动入场权限（车位先到先得）
- 切换 Primary 车辆时，所有车辆必须不在场
- 原 Primary 车辆不能有未完成的入场申请

## 3. Visitor 车辆申请

### 3.1 申请流程

1. 进入「Visitor 申请」页面
2. 选择已绑定的车牌
3. 填写申请原因（如「亲属来访」）
4. 点击「提交申请」
5. 等待物业审批

### 3.2 审批结果

- 审批通过后，系统发送订阅消息通知
- 通过后有 24 小时激活窗口，Visitor 车辆需在此期间首次入场
- 超过 24 小时未入场，授权自动取消

### 3.3 Visitor 车辆使用规则

- 首次入场即激活，之后可多次进出
- 累计停放时长上限 24 小时
- 每个房屋号每月 Visitor 配额 72 小时
- 超时后系统自动提醒业主和物业

## 4. 月度配额查询

1. 进入「配额查询」页面
2. 查看当月已使用时长和剩余时长
3. 进度条显示配额使用比例
4. 使用量达到 60 小时时系统发送提醒

**配额说明：**
- 每月 1 日自动重置
- 总配额 72 小时 / 月 / 房屋号

## 5. 入场记录查询

1. 进入「入场记录」页面
2. 选择查询时间范围（必填）
3. 下拉刷新获取最新数据
4. 上拉加载更多历史记录

**显示信息：** 车牌号、入场时间、出场时间、停放时长

## 6. 常见问题

**Q: 注册后多久能使用？**
A: 提交注册后需等待物业审核，审核通过后系统会发送通知，届时即可登录使用。

**Q: 为什么无法设置 Primary 车辆？**
A: 请确认所有车辆均已出场，且原 Primary 车辆没有未完成的入场申请。

**Q: Visitor 申请被拒绝怎么办？**
A: 可查看驳回原因，修正后重新提交申请。

**Q: 为什么显示「配额已用完」？**
A: 每个房屋号每月 Visitor 配额为 72 小时，当月配额用完后需等下月 1 日重置。

**Q: 车位满了怎么办？**
A: 车位采用先到先得机制，Primary 车辆和 Visitor 车辆共享车位池。车位满时无法入场，请稍后再试。

**Q: 如何修改手机号等个人信息？**
A: 进入「个人信息」页面提交修改申请，物业审批通过后自动更新。
